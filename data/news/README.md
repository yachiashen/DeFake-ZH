## news 資料夾說明

請將 **NewsCrawler** 專案的新聞資料放置於 ``csv`` 資料夾下。  
格式如下

```python
news
├── csv
│   ├── fake                # NewsCrawler 專案中的假新聞資料夾
│   ├── news                # NewsCrawler 專案中的新聞資料夾
│   └── news_verification   # NewsCrawler 專案中的驗證資料夾
└── json                    # csv 格式新聞 → json 格式新聞 
```
